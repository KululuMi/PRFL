# PRFL: Process Reward Feedback Learning

Demo page for PRFL (Process Reward Feedback Learning) - Video Generation Models are Good Latent Reward Models.

## GitHub Pages Setup

This repository is configured to be hosted on GitHub Pages. The main page is `index.html`.

### Access the Live Site

Once GitHub Pages is enabled, the site will be available at:
- `https://kululumi.github.io/PRFL/`

### How to Enable GitHub Pages

1. Go to your repository: https://github.com/KululuMi/PRFL
2. Click on **Settings** (仓库设置)
3. Scroll down to **Pages** section (在左侧菜单中)
4. Under **Source**, select **Deploy from a branch**
5. Choose **main** branch and **/ (root)** folder
6. Click **Save**
7. Wait a few minutes for GitHub to build and deploy your site
8. Your site will be available at the URL shown in the Pages section

### Local Development

Simply open `index.html` in a web browser to view the page locally.
